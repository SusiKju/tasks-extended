# Brainstorming

Hier kannst du frei reinschreiben was dir an Ideen, To-Dos oder Themen einfällt,
ohne dass es direkt ein Task wird. Wird bei jedem Bootstrap geprüft.
Wenn hier Inhalt steht, fragt der Skill einmal nach ob er die Punkte zu echten
Tasks ausarbeiten soll.

Format: einfach reinschreiben, beliebig formatiert. Der Inhalt wird selbst
erkannt und in Tasks aufgegliedert.
